Reflections

May 24, 2015

Version control records changes made to files and allows you to revert back to changes at anytime. It is great for comparing files, when modifications were made and by who. It is great for accountability between multiple users on a project. It is essentially like backing up a project anytime and if a mistake was made, the project's work is still around so the hours spent working on it won't completely go to waste. If you try typing 'git log' in your terminal, it will show a log of all the changes made, by who and when. Github is the icing between users. This web app helps facilitate the work on a project and can be viewed and forked by anyone. This is a collaborative app that does wonders. Give it a try!
